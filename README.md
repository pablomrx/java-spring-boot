#  Voll.med API - Spring Boot 3 ☕

Repositório com exercícios e desafios desenvolvidos durante os cursos **"Spring Boot 3: desenvolva uma API Rest em Java"**, **"Spring Boot 3: aplique boas práticas e proteja uma API Rest"** e **"Spring Boot 3: documente, teste e prepare uma API para o deploy"** da Alura.

---

## 📖 Conteúdos estudados

- Criação de APIs REST com Spring Boot 3
- Orientação a Objetos aplicada ao domínio da aplicação
- Padrão MVC
- Controllers, Services e Repositories
- Spring Data JPA
- Hibernate
- Persistência de dados com MySQL
- Migrations com Flyway
- Validações com Bean Validation
- Paginação e ordenação de resultados
- DTOs e Records
- Tratamento global de exceções
- Spring Security
- Autenticação e autorização
- JWT (JSON Web Token)
- Criptografia de senhas com BCrypt
- Filtros de segurança
- Perfis de ambiente (dev, test e prod)
- Documentação da API com Swagger/OpenAPI
- Testes automatizados com JUnit 5
- Mocks com Mockito
- Testes de controllers com MockMvc
- Testes de repositórios com DataJpaTest
- Regras de negócio para agendamento e cancelamento de consultas
- Boas práticas de organização do código
- Preparação da aplicação para deploy

### 🧑 Pacientes

- Cadastro de pacientes
- Atualização de dados cadastrais
- Listagem com paginação
- Exclusão lógica de registros

### 📅 Consultas

- Agendamento de consultas
- Escolha automática de médicos por especialidade

- Restrição de horário de funcionamento da clínica

- Cancelamento de consultas
- Validação de antecedência mínima para cancelamento

## 🎯 Objetivo

Praticar o desenvolvimento de APIs REST utilizando **Spring Boot 3**, aplicando conceitos de **Orientação a Objetos**, persistência de dados com **JPA/Hibernate**, autenticação com **JWT**, documentação com **Swagger/OpenAPI** e testes automatizados com **JUnit e Mockito**, seguindo boas práticas utilizadas em aplicações Java modernas.
